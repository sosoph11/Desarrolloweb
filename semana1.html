Desarrollo Web

Desarrollo Web: Crea sistemas accesibles desde cualquier dispositivo sin instalaciones previas, usando tecnologías web y hardware/software básicos.

Desarrollo Web vs. Diseño Web

Diseño Web: Enfocado en apariencia, navegación y experiencia del usuario.

Desarrollo Web: Se encarga de la funcionalidad, como backend, gestión de contenido y e-commerce.

Arquitectura de Software

Diseño estructural de un sistema de software que define cómo interactúan sus partes.

Arquitectura Cliente-Servidor

Cliente solicita servicios al servidor que responde a las peticiones. El servidor maneja peticiones en un puerto específico, y el cliente usa un puerto libre. Servidores pueden ser simples o manejar peticiones concurrentes y seguridad avanzada.

Partes del Sistema:

  • Cliente: Programa que realiza solicitudes y espera respuestas (e.g., navegador web).
  • Servidor: Programa que ofrece servicios en red, maneja solicitudes y envía respuestas. Puede ser sencillo o complejo, con requerimientos de seguridad y protección.

Roles y Etapas en el Desarrollo Web

Desarrollador/a Frontend: Programa la lógica del lado del cliente, maneja HTML, CSS, JS, y frameworks como React y Angular. Se encarga de la interfaz y la interacción con APIs.

Desarrollador/a Backend: Programa la lógica del lado del servidor, gestiona la base de datos y maneja lenguajes y frameworks del servidor como NodeJS y Django. Se ocupa de la lógica del negocio de la aplicación.

Diseñador/a UI-UX: Se enfoca en el diseño visual y la experiencia del usuario, utilizando herramientas como Adobe XD y Figma. Trabaja con equipos de marketing y diseño gráfico.

Desarrollador/a FullStack: Puede cubrir roles tanto de frontend como de backend y administración de bases de datos, aunque no suele abarcar diseño gráfico.

Etapas del Desarrollo de Software

  1. Planificación: Documentación de requisitos, tecnologías, diagramación de entidades y relaciones, y definición de plazos y hitos.
  2. Diseño: Definición de flujos de usuario, diseño de la interfaz gráfica, y maquetado con la gama de colores y estilo definidos.
  3. Desarrollo: Construcción de bases de datos, programación de la lógica del cliente y servidor, desarrollo de la interfaz y su funcionalidad. Es la etapa más extensa y requiere coordinación.
  4. Despliegue: Colocación del producto en producción tras las pruebas y verificación de que cumple los requisitos.
  5. Soporte: Mantenimiento continuo, corrección de errores, implementación de cambios menores y mantenimiento de la base de datos.

HTML y CSS

HTML (HyperText Markup Language): Es el lenguaje fundamental para crear y estructurar contenido en una página web. Define el contenido y los enlaces entre páginas, pero no su funcionalidad.

CSS (Cascading Style Sheets): Es el lenguaje utilizado para describir la presentación de documentos HTML. Define cómo se deben mostrar los elementos en pantalla, incluyendo estilos y diseño visual.

Editor de Texto

Programa para trabajar con archivos de texto plano. Se recomienda Visual Studio Code.

Herramienta del Desarrollador Web

Esta herramienta permite ver detalles ocultos en la web, como el HTML, los estilos visuales y el código JavaScript. Las principales pestañas son:

  • Elements (Elementos): Muestra el HTML y representación visual.
  • Console (Consola): Ejecuta código JavaScript.
  • Sources (Fuentes): Muestra recursos de la página (HTML, JS, CSS).
  • Network (Red): Muestra el tráfico de red.

Incluir JavaScript y CSS

JavaScript: <script src="archivo.js"></script> (en <head> con defer o al final del <body>).

CSS: <link rel="stylesheet" href="archivo.css"> (en <head>).

Sitios de Ayuda

  • Stack Overflow: Resuelve errores y problemas comunes con soluciones votadas por la comunidad.
  • JS Fiddle: Herramienta en línea para probar HTML, CSS y JavaScript, con opciones para experimentar y compartir resultados.
  • W3Schools: Recursos para aprender HTML, CSS, JavaScript con explicaciones y ejemplos prácticos.
