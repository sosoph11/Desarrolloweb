CSS(Cascading Style Sheets)

Es un lenguaje que permite separar el contenido de una página web (HTML) de su diseño o formato. En las primeras versiones de HTML, el contenido y el diseño estaban mezclados en el código fuente, pero actualmente se pueden manejar por separado usando CSS para definir el estilo.

Hay tres maneras de dar estilo a los elementos en una página web:

  1. Style en HTML: Afecta solo al elemento en el que está incluido.
  2. Style en HTML: Afecta a los elementos seleccionados en toda la página web.
  3. Hoja de estilos externa(.css): Se vincula en el archivo HTML mediante la etiqueta link y define el estilo de presentación (color, tamaño, fuente, bordes, márgenes, posición, etc.) para todas las etiquetas.

Utilizar hojas de estilo permite un diseño consistente y fácil de modificar. Si varias páginas web usan la misma hoja de estilo, basta con modificar esta para cambiar la apariencia de todas las páginas que la referencian.

Estilo en cascada

El formato final de un elemento en CSS depende de la combinación de todas las propiedades que lo afectan, lo que se denomina "estilos en cascada". Esto significa que un elemento hereda las propiedades de los elementos que lo contienen. Por ejemplo, el color de texto de un párrafo <p> puede estar determinado por la propiedad establecida para el elemento <body>.

Se pueden definir varios selectores simultáneamente escribiéndolos separados por comas, pero no es correcto definir varias propiedades simultáneamente.

Una hoja de estilo puede incluir comentarios, los cuales se delimitan con /* ... */, similar a los comentarios en el lenguaje de programación C.

En las hojas de estilo CSS, los selectores son la parte de las reglas que indican al navegador a qué elementos deben aplicarse las propiedades especificadas en las declaraciones.

Estilos en cascada

Las hojas de estilo se llaman "en cascada" porque las propiedades de estilo pueden estar escritas en varios lugares y afectar a diferentes elementos. Las propiedades definidas para un elemento contenedor también se aplican a los elementos contenidos. Además, varias reglas pueden aplicarse a un mismo elemento. Si no hay conflictos entre las propiedades, el navegador aplica todas las propiedades. Si hay conflictos, existen reglas para determinar qué propiedad tiene preferencia.

Reglas de aplicación de los estilos:

  1. Atributo style > Etiqueta <style> > Hoja de estilo enlazada > Propiedades por defecto del navegador.
  2. Etiqueta <style> > Hoja de estilo enlazada > Propiedades por defecto del navegador.
  3. Hoja de estilo enlazada > Propiedades por defecto del navegador.
  4. Si no se aplica ningún estilo específico, el navegador usa sus propiedades por defecto.
  5. En conflicto entre diferentes hojas de estilo, se aplica el valor de la última hoja de estilo enlazada (último <link> en <head>).

Tamaños y distancias

La propiedad width define el ancho de un elemento usando un valor numérico (con unidad) o un porcentaje (respecto al elemento padre). Si no se especifica, el elemento ocupa todo el ancho disponible.

La propiedad height define la altura de un elemento con un valor numérico (con unidad), pero no en porcentaje. Si no se especifica, el elemento ocupa el espacio necesario para su contenido. Si se establecen tanto width como height, el contenido puede no caber y se debe usar la propiedad overflow para manejar este caso.

Tamaño de imágenes:

  • Las propiedades width y height también se aplican a imágenes, prevaleciendo sobre los atributos width y height en la etiqueta <img>.
  • Si se indica solo una propiedad (ancho o altura), la imagen mantiene la proporción.
  • Si se indican ambas propiedades, la imagen se ajusta al tamaño especificado.

Unidades relativas

Las unidades relativas se dividen en dos tipos: aquellas que se basan en el tamaño del tipo de letra y aquellas que se basan en las dimensiones de la pantalla.

Unidades relativas respecto al tipo de letra:

  • em: Tamaño de fuente del elemento contenedor (1em = 100%).
  • ex: Altura de la letra "x" del elemento contenedor.
  • ch: Altura de la cifra "0" del elemento contenedor.
  • rem: Tamaño de la fuente del elemento raíz de la página.

Las más utilizadas son em y rem.

Unidades relativas respecto a la pantalla:

  • vw: el 1% del ancho de la ventana (sin contar bordes, pestañas, barras de desplazamiento, etc.).
  • vh: el 1% del alto de ventana (sin contar bordes, pestañas, barras de desplazamiento, etc.).
  • vmin: el menor de los tamaños vw y vh.
  • vmax: el mayor de los tamaños vw y vh.
